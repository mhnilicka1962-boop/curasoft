Stand: 25.02.2026

## Zielgruppen

| Rolle | Fokus |
|---|---|
| **Admin** | Alles: Klienten, Planung, Rechnungen, Mitarbeiterverwaltung, Stammdaten |
| **Pflege** | Einsätze, Vor-Ort-Check-in/out, Rapport schreiben |
| **Buchhaltung** | Rechnungen, PDF/XML-Export, Bexio-Sync |

---

## Empfohlene Schulungsform: Kombiniert

### Option A — Screencast-Videos (Kernstück)
- Kurze Videos pro Thema (3–5 Min.)
- Direkt in der App unter **Hilfe** verlinkt
- Pflegepersonen können jederzeit nachschauen — kein Termindruck
- Neue Mitarbeiter schulen sich selbst
- Tool: **Loom** (kostenlos, direkt im Browser aufnehmen) oder OBS

### Option B — Live-Einführung (einmalig, 60 Min.)
- Gemeinsame Einführung für das ganze Team
- Demo auf **curasoft.ch** mit Testdaten
- Danach Videos als Referenz

### Empfehlung: A + B kombiniert
1. **30 Min. Live** — Login, Navigation, erstes Gefühl
2. **Videos** als Nachschlagewerk für Details
3. **Hilfe-Seite** in der App für schnelle Fragen

---

## Schulungsreihenfolge

### Block 1 — Für alle (15 Min.)
1. Login per Magic Link (E-Mail-Link)
2. Passkey / Face ID einrichten
3. App als Icon auf Homescreen speichern
4. Navigation kennenlernen

### Block 2 — Pflege (20 Min.)
1. Eigene Einsätze anzeigen
2. Vor-Ort: Check-in / Check-out
3. Checkliste ausfüllen
4. Rapport schreiben — Diktat + KI Bericht schreiben

### Block 3 — Admin (30 Min.)
1. Neuen Klienten anlegen (Adresse, Krankenkasse, Region)
2. Einsatz planen
3. Mitarbeiter anlegen + Einladungsmail
4. Rechnungen erstellen, PDF + XML exportieren

### Block 4 — Buchhaltung (15 Min.)
1. Rechnungsübersicht + Status-Workflow
2. PDF herunterladen / drucken
3. XML 450.100 exportieren (für MediData / Krankenkasse)
4. Bexio-Sync (falls konfiguriert)
