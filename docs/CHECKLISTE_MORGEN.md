# Checkliste für morgen — Mitarbeiterin + Angehörigenpflege

## 1. Neue Mitarbeiterin wird erfasst

### In Spitex

| Schritt | Wo | Was |
|---------|-----|-----|
| 1. Anlegen | `/mitarbeiter` → „+ Neu“ | Anrede, Vorname, Name, E-Mail*, Rolle* (admin/pflege/buchhaltung), evtl. Pensum, Eintrittsdatum |
| 2. Speichern | — | **Einladungs-Mail wird automatisch versendet** (Link gültig 48h) |
| 3. Mitarbeiterin | E-Mail öffnen → Link klicken | Passwort setzen (min. 8 Zeichen, bestätigen) |
| 4. Login | `/login` | Mit E-Mail + neuem Passwort anmelden |
| 5. Stammdaten | Mitarbeiter-Detail → Stammdaten | Adresse, AHV, IBAN, Telefon etc. nachtragen |
| 6. Qualifikationen | Mitarbeiter-Detail → Qualifikationen | Ausbildung (z.B. HF, DN I, PA, FAGE) zuweisen |
| 7. Klient-Zuweisung | Mitarbeiter-Detail → Zugewiesene Klienten | Bei Bedarf Klienten mit Rolle (Hauptbetreuer/Betreuer) zuweisen |

### Externe Schritte (Behörden, SPITEX)

| Thema | Zuständigkeit | Hinweis |
|-------|----------------|---------|
| **Berufsausübungsbewilligung** | Kantonale Gesundheitsdirektion | Pflegefachpersonen benötigen Bewilligung; Gesuch durch Arbeitgeber, ca. 2 Monate vor Tätigkeitsbeginn |
| **Betriebsbewilligung SPITEX** | Kantonale Gesundheitsdirektion | Die SPITEX-Organisation braucht eine gültige Betriebsbewilligung |
| **Anstellung** | Arbeitgeber | Vertrag, AHV, Lohnabrechnung etc. |

**Kanton Aargau:** [ag.ch – SPITEX Betriebsbewilligung](https://ag.ch/de/themen/gesundheit/gesundheitsberufe/betriebsbewilligungen/spitex)

## 2. Person, die einen Angehörigen pflegt (Angehörigenpflege)

### Zwei Fälle unterscheiden

| Fall | Bedeutung | In Spitex |
|------|-----------|-------------|
| **A: Angehöriger als Kontakt** | Familie/Bekannte als Kontaktperson des Klienten (nicht pflegend tätig) | Klient-Detail → Kontakte & Angehörige → „+ Kontakt“ mit Rolle z.B. „Angehöriger“, „Rechnungsempfänger“, „Bevollmächtigt“ |
| **B: Pflegender Angehöriger** | Die Person wird von der SPITEX angestellt und pflegt den Angehörigen gegen Lohn | **Als Mitarbeiter erfassen** (wie oben) + Klient-Zuweisung beim Klienten, den sie pflegt |

### Fall B: Pflegender Angehöriger — Ablauf

1. **Mitarbeiter anlegen** (Stammdaten, Einladung, Passwort, Qualifikationen)
2. **Klient-Zuweisung**: Auf Mitarbeiter-Detail → Klient hinzufügen (z.B. Rolle „Betreuer“ oder „Hauptbetreuer“)
3. **Einsätze planen**: Wie bei jeder anderen Pflegeperson — Einsatz mit diesem Mitarbeiter anlegen
4. **Regelungen beachten**: Kantonale Vorgaben (Lohn, Vertrag, evtl. Bewilligung)

### Regelungen Schweiz (Angehörigenpflege)

| Punkt | Details |
|-------|---------|
| **Seit 1.5.2023** | Freunde, Bekannte und Familienmitglieder können Pflege ausführen, wenn mit SPITEX eine Zusammenarbeit vereinbart wird |
| **Anstellung** | Pflegende Angehörige können bei SPITEX angestellt werden (fairer Lohn) |
| **Kantonal** | Jeder Kanton hat eigene Tarife, Verträge, Bewilligungen |
| **Finanzen** | Möglich: Pauschalierter Tageslohn (20–30 CHF), Betreuungsgutschriften, Hilflosenentschädigung etc. |

**Wichtig:** Zuständig sind die kantonale Gesundheitsdirektion und die SPITEX-Organisation. Spitex bildet nur die Planung und Abrechnung ab — rechtliche/behördliche Schritte müssen ausserhalb der Software erfolgen.

## 3. Übersicht: Klient-Kontakte vs. Mitarbeiter

| | KlientKontakt (Angehöriger) | Benutzer (Mitarbeiter) |
|---|----------------------------|-------------------------|
| **Zweck** | Kontaktperson, evtl. Rechnungsempfänger, Bevollmächtigter | Pflegeperson, die Einsätze ausführt |
| **Login** | Nein | Ja |
| **Einsätze** | Kann keine planen/abrechnen | Kann Einsätze erhalten und abarbeiten |
| **Wo erfassen** | Klient-Detail → Kontakte | Stammdaten → Mitarbeitende |
| **Rollen** | Angehöriger, gesetzl. Vertreter, Rechnungsempfänger, Notfallkontakt | admin, pflege, buchhaltung |
