# CuraSoft — Ihre persönliche Demo

Guten Tag

Herzlich willkommen bei CuraSoft. Ich habe Ihnen eine Demo-Umgebung vorbereitet — einfach auf die Links klicken, kein Passwort nötig.

---

## Am Computer — Verwaltung

**→ https://www.curasoft.ch/demo/admin**

Nach dem Klick sehen Sie das Dashboard mit allen heutigen Einsätzen. Hier ein kurzer Rundgang:

**Klienten** — öffnen Sie z.B. Elisabeth Brunner: Adresse, Krankenkasse, Arzt, Diagnosen und die nächsten 14 Tage Pflegeplan auf einen Blick.

**Tourenplanung** — alle Touren des Tages, pro Mitarbeiterin sortiert nach Uhrzeit. Tour-Detail zeigt Check-in/out-Zeiten und die optimierte Route auf der Karte.

**Einsatzplanung (Kalender)** — alle Mitarbeitenden als Zeilen, Einsätze als farbige Balken. Doppelbelegungen werden automatisch rot markiert. Per Drag & Drop verschieben oder umteilen.

**Rechnungsläufe** — klicken Sie auf einen der vier Läufe. Sie sehen alle Rechnungen pro Klient mit Patientenanteil und Kassenanteil. Pro Rechnung: PDF mit Swiss QR-Zahlteil und XML 450.100 für die Krankenkasse — alles auf Knopfdruck.

## Am Smartphone — Pflege vor Ort

Öffnen Sie einen der folgenden Links auf dem Handy:

| Mitarbeiterin | Link |
|---------------|------|
| Sandra Meier | https://www.curasoft.ch/demo/pflege |
| Peter Keller | https://www.curasoft.ch/demo/peter |
| Anna Brunner | https://www.curasoft.ch/demo/anna |

Nach dem Klick sehen Sie direkt die heutige Tour. Tippen Sie auf einen Klienten — Adresse mit Maps-Link, Telefon anrufbar, Notfallnummer, Diagnosen. Dann **Check-in**, Leistungen abhaken, Rapport schreiben (oder per KI aus Stichworten generieren lassen), **Check-out** — fertig.

**Tipp:** Safari → Teilen-Symbol → «Zum Home-Bildschirm» — die App erscheint als Icon, ab sofort mit Face ID einloggen.

## Hinweis

Dies ist eine gemeinsame Demo-Umgebung — bitte keine echten Patientendaten eintragen.

Für den produktiven Einsatz erhalten Sie eine eigene Instanz unter Ihrer Adresse (z.B. `ihrespitex.curasoft.ch`) mit vollständig getrennter Datenbank — Ihre Daten gehören nur Ihnen.
