### 2.1 Tour heute
- Nach Login: direkt auf der heutigen Tour

### 2.2 Vor-Ort beim Klienten
- Auf **Elisabeth Brunner** tippen
- Adresse mit Maps-Link, Telefon anrufbar, Notfallnummer rot
- Diagnosen + Verordnung auf einen Blick
- **▶ Check-in** tippen → Uhrzeit wird erfasst

### 2.3 Leistungserfassung
- Leistungen abhaken: Körperpflege, Medikamente, Mobilisation usw.
- Minuten werden automatisch summiert

### 2.4 Rapport schreiben
- **+ Rapport** tippen
- Text eintippen oder diktieren
- KI-Button: aus Stichworten automatisch vollständigen Pflegebericht erstellen
- Speichern → Rapport sofort für Admin sichtbar

### 2.5 Check-out
- **■ Check-out** → Einsatz abgeschlossen
- Dauer gespeichert, bereit für Abrechnung

## Häufige Fragen

**Wie lange dauert die Einrichtung?**
→ Startklar in 15 Minuten: Firma, erste Klienten, erste Mitarbeitende.

**Was kostet es?**
→ Preis je nach Anzahl Klienten / Mitarbeitende — individuelles Angebot.

**Eigene Instanz?**
→ Ja — eigene Adresse (z.B. `ihrespitex.curasoft.ch`), eigene Datenbank, Ihre Daten gehören nur Ihnen.

**KVG-konform?**
→ Ja — XML 450.100, Swiss QR-Zahlteil, Tiers garant/payant, ICD-10 Diagnosen.
